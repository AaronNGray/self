Recap

  • Literals: integers, floats, strings

  • Messages: unary, binary keyword. Resends

  • Literal objects: slot initializers

  • Methods: method is prototype for activation.

  • Blocks: block is object with value method and lexical scoping. Non-local returns

  • Primitives: syntactically distinct

  • Mirrors: reflection

  • Annotations: comments, declarations, module info.
