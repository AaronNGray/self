Setting the color:


The finished item:


A final touch is to harmonize all the colors. To do this we get a color changer from the `Change Color...' right-button menu on our UI object, and use the sliders to set a color. The color applies to the whole object. Morphs that need to distinguish themselves from the background color (such as labels) set an appropriate complementary color. We can also adjust layout parameters and insert spacers to get the desired look.
