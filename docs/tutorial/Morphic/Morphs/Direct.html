Direct manipulation of morphs

All morphs share the same right-button (blue) menu, which lets the user perform various morph-related operations. We will look at the other entries later, but for now let's consider these:

    Grab
    Some morphs are mouse-sensitive (e.g., buttons) and clicking with the left-mouse button does not pick them up but activates the morph. This menu item lets the user pick up even mouse-sensitive morphs.

    Duplicate
    duplicates the morph. If the morph is composite, the whole structure is duplicated. The copy is put in the hand.

    Outliner for Morph...
    The slot-level representation of objects we have seen are also known as outliners. This item summons the outliner for the object implementing the morph.

    Move to Own Window
    Creates a new morph world, moves the morph to that world, and opens a window on that morph on every user's screen.

    Dismiss
    Deletes the morph from this world.
