Manipulating submorphs

Submorphs menu:

Manipulating the submorph structure

The submorph structure can be directly manipulated from the user interface. On the blue menu:

Submorphs Summons a menu of the submorphs below the hand.

Embed in morph below Embed the morph directly below the hand into a morph below. If there is more than one candidate morph, a menu lets the user choose.

Core sampler Creates a tool useful for exploring and manipulating the submorph structure (see below). Can do both of the things above.
