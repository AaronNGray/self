The Self Virtual Machine

The Self Group

How the Self VM works inside, really.

The purpose of this document is to explain how the Self VM is structured. It should be read as a prelude to doing work on the VM. Actually, many parts of the VM are not heavily related, so you may get by just reading some sections. Also, by its very nature, this document will probably be inaccurate, incomplete, and out-of-date. But hopefully it will still be worth reading. There are other things you should read first, though. You need to have some familiarity with Self. Also, read the various published papers on the Self implementation (in particular, [CUL89], which will you give you a good overview of the structure of the VM, [HCU91] (pics), [Höl94] (sic), [HCU92] (deoptimizing), [CU89], [CU90], [CU91], [Cha92] new compiler, ....
